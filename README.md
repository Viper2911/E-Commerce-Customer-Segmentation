# 🛍️ E-Commerce Customer Segmentation AI

## 🎯 Project Overview
This project applies Unsupervised Machine Learning to categorize an e-commerce platform's customer base into distinct behavioral segments. By replacing generic email blasts with highly targeted marketing campaigns based on these AI-driven segments, businesses can optimize their ad spend and drive targeted conversion increases.

The core engine uses **RFM (Recency, Frequency, Monetary)** feature engineering combined with a **K-Means Clustering** algorithm to automatically group customers based on their historical purchasing patterns.

## ✨ Key Features
* **Automated RFM Engineering:** Transforms raw transaction logs into actionable behavioral metrics, applying Log-Transformations to handle severe revenue skewness.
* **K-Means Clustering:** Mathematically divides the customer base into 4 highly distinct, actionable marketing personas.
* **Interactive Streamlit Dashboard:** Allows marketing managers to input a user's transaction history and instantly receive their segment classification and a recommended marketing action.

## 📊 The 4 Marketing Segments
Based on the cluster centers generated by the model, customers are mapped to:
1. **🌟 Loyal VIPs:** High spend, high frequency, recent engagement. *(Action: Premium loyalty rewards, no heavy discounting).*
2. **🛒 Recent Shoppers:** Active right now, but low historical spend. *(Action: Nurture with related product recommendations).*
3. **⚠️ Churn-Risk:** High historical spend, but haven't visited in months. *(Action: Aggressive 'win-back' discount codes).*
4. **📉 Lost Customers:** Low spend, single purchase, inactive for 6+ months. *(Action: Standard automated newsletters, minimal ad spend).*

## 📈 Model Evaluation
Because this uses unsupervised learning, traditional accuracy metrics do not apply. The mathematical distinctness of the clusters was evaluated using:
* **Silhouette Score:** ~0.33 (Indicating valid cluster boundaries despite the natural overlap in e-commerce purchasing behavior).
* **Davies-Bouldin Index:** ~1.01 (Demonstrating strong separation between cluster centers).
